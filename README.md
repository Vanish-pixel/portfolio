# Portfolio — Valentin

My personal portfolio website. Training as an IT specialist for system integration —
interested in servers, networks, Linux, GitHub and software development.

Built from scratch with **HTML, CSS and vanilla JavaScript** — no frameworks, no build
step. Designed to be fast, accessible and responsive across desktop, tablet and mobile.

## ✨ Features

- Modern, minimal dark theme with glassmorphism
- Smooth animations, scroll reveal and a typing effect in the hero
- Sticky navbar with active-section highlighting + mobile menu
- Custom cursor, scroll progress bar and back-to-top button
- Fully responsive and accessible (skip link, ARIA, reduced-motion support)
- SEO-ready (meta tags, Open Graph, structured data)

## 📂 Structure

```
index.html   → markup and content
style.css    → design system, layout and animations
script.js    → interactivity (typing, nav, reveal, cursor, form)
```

## 🚀 Deploy on GitHub Pages

1. Go to **Settings → Pages**.
2. Under **Source**, choose branch `main` and folder `/ (root)`.
3. Save — the site goes live at `https://vanish-pixel.github.io/portfolio/`.

## 🛠️ Customize

- Text content lives in `index.html` (search for the `EDIT:` comments).
- Colors and fonts are CSS variables at the top of `style.css` (`:root`).
- Typing words are in the `CONFIG` object in `script.js`.
- Replace the `#` placeholder links with your real GitHub, Discord and LinkedIn URLs.
